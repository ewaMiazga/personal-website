# 3DGStream Project Page

Edit `index.html` for the project page content.

Project-specific media lives here:

- `banner/` - social preview/banner image.
- `results/` - result figures used in the evaluation sections.
- `teaser-videos/` - side-by-side teaser videos.
- `carousel-videos/` - comparison videos used in the horizontal video section.
- `unused-videos/` - kept videos that are not currently referenced by the page.

Shared site files live two folders up:

- `../../static/css/project-page.css`
- `../../static/js/index.js`
- `../../assets/epfl-logo.png`
